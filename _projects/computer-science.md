[[personal/Уроки педагогики/технологии]]:

1. markdown позволит без усилий превращать полученную информацию в знания работая за компьютером;
2. Повторяя полученные на уроках математики и химии формулы, вводя их в компьютер, используя язык разметки LaTeX учащийся углубляет понимание;
3. Диаграммы mermaid лёгкий способ создания диаграммы и схематических изображений для уроков истории, информатики, экономики, при ответе можно использовать электронную доску и проектор;
4. Умение работать с источником информации, указание библиографической ссылки позволит вернуться к источнику в чтобы лучше обдумать сказанное и избежать плагиата при защите проекта;
5. reveal.js позволяет делать презентации из конспектов markdown используя одну команду деления на слайды --- это экономит время и позволяет сосредоточиться на учебной задаче;
6. Obsidian объединяет всё вышеперечисленные технологии и стандарты в одной программе, позволяя создавать свою индивидуальную базу знаний в парадигме [[personal/index]]
7. Github открывает возможность совместной работы над проектами
